Code Features

Resposive

I implemented a responsive code using only display property block and inline-block.

Stable

The main page, index.html, is the only one to be loaded. Other contents apears in an embeded element preventing reloads of the entire website on each click. This aproach avoid blinks while browsing and reduce the amount of data transferig through the net.

The menu behavior was set up with JavaScript. There is an object storing the menus, as properties, and storing another object with 2 properies each one: the id and src attribute.

A for...in statement iterates over the object and add an event listener for each menu.

Before changing src propery, it turns the content transparent and gradually fade it in. This creates a soft effects to the transitions.

Constrast

The colors ware checked with Contrast Checker and distinguish all the elements.

Whatsapp

The way of contact is whatsapp, not email. JavaScript open the app with a link created with the address string concatenated with the message wrote in text field.
