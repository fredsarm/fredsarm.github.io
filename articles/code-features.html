Code Features

Resposive

I implemented a responsive code using only display property block and inline-block. Certainly, there is more space to improve it.

Rendering Stability

The main page, index.html, is the only one to be loaded. Other contents appear in an embedded element preventing reloads of the entire website on each click. This approach avoid blinks while browsing and reduce the amount of data transferring through the net.

The menu behavior was set up with JavaScript. There is an object storing the menus, as properties, and storing another object with 2 properties each one: the id and src attribute.

A for...in statement iterates over the object and add an event listener for each menu.

Before changing src property, it turns the content transparent and gradually fade it in. This creates a soft effects to the transitions.

Contrast

The colors were checked with Contrast Checker and distinguish all the elements.

WhatsApp

The way of contact is WhatsApp, not email. JavaScript open the app with a link created with the address string concatenated with the message wrote in text field.
